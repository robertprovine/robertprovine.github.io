Project Portfolio

Beast

Live Github

A classic DOS arcade game from the 80's, reprogrammed for the modern web browser. Move blocks to crush the enemy ‘H’ beasts before they make a meal out of you. Challenges included creating algorithms that optimized Beast traversal to catch the player. Written in Javascript.

Data-Bot

Github

A CRUD scaffolding application for Express.js and the ejs view templating engine. Create data models using a simple web based UI, then download a generated Express.js/ejs project folder that follows a MVC design pattern similar to a Ruby on Rails application. Written in ES6 using Webpack/Babel.

Reaktormap

Github

A sample map (.map) generator for Native Instruments Reaktor Software. This is a tool to make loading sample maps into drum-machines much easier. The file format specification for Reaktor sample maps is proprietary, so reverse engineering of the format was necessary using a hex editor. Written in C/C++ using POSIX system calls

Hypertext-Control

Github

A MIDI controlled HTML/CSS generator. Plug in the Behringer X-Touch MIDI controller via USB, load up the web app and start pushing buttons and turning knobs to change CSS styles. In minutes, you'll have a mockup webpage.
